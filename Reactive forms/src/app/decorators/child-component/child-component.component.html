In this example I have two components. The parent component and the child component. The child component is inside the parent component HTML code and it is called with the selector.
The parent component has a variable defined "message". It will contain the text that will be sent inside the selector child just like this:

[messageToShow]="message"

Note:
- "message" is the variable that contains the text.

- "messageToShow" is the variable used to recieve this information in the child component.


The component child will get its value with the decorator @input in its ts component code, just like this: @Input() messageToShow: any;


To show the content of the variable we use interpolation like this {{interpolation}}

{{message1ToChild}}

{{message2ToChild}}
